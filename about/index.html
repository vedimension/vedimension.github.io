访客你好，我是 VE，人間開花（HUMAN BLOOM）是我的博客名。源自日本摇滚乐队 RADWIMPS 同名专辑。

搭建博客的初衷是为了满足自己的表达欲，有时会突然想写些什么但又找不到合适的地方。一开始并没有找到符合我需求的平台，定制化程度不高，也不符合审美，所以开始自己搭建起了静态博客，从 WordPress 折腾到 Jekyll/Hexo 这几年换过不少。

2018 年之前写的大部分文章已经隐藏。

现在的博客是在 2021 年从旧版迁移的过来的新页面，使用了兰迪博客主题，在原版基础上自己修改了部分样式，博客依赖 Github Pages 托管。

主页的图片是我非常喜欢的日本插画师青藤スイ的作品。主页的一小段诗节选自 R.S 托马斯的 Reflections，同时也是游戏《极乐迪斯科》开篇的引言。

目前职业是前端工程师，博客内容非常个人生活向，并非主写技术文章。喜欢唱歌，偶尔会上传些翻唱作品到网易云上。我的微博是@夜間開花，我的推特是@bouquetrender。

博客所有文章采用「非商业性使用-禁止演绎 4.0 国际许可协议 CC BY-NC-ND 4.0」，但好像并没有什么值得搬运的内容。博客文章结尾处添加了 Cusdis 评论系统，同时也可以通过邮箱 miyashitasakuya@gmail.com 联系我。

最后非常感谢访问人間開花。
